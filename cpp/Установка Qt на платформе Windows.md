**Шаг 1**. [Скачать](https://www.msys2.org/) установщик среды Linux MSYS2 с официального сайта.
**Шаг 2**. Запустить скачанный исполняемый файл и следовать шагам установщика.
**Шаг 3**. Запустить терминал MSYS2, который будет расположен по пути:

```cmd
C:\msys64\msys2.exe
```

**Шаг 4.** Обновить базы данных доступных пакетов командой:

```bash
pacman -Syu
```

В процессе обновления баз данных может быть предложено установить отсутствующие компоненты или обновить их до актуальной версии, **соглашаемся**. 

> После завершения установки обновлений необходимо закрыть терминал и открыть его заново.  

**Шаг 5.** Для установки среды разработки выполните команду:

```bash
pacman -S base-devel git mingw-w64-x86_64-toolchain mingw-w64-x86_64-cmake mingw-w64-x86_64-qt5-static mingw-w64-x86_64-qt-creator
```

> В процессе установки соглашаемся на все вопросы. Если необходимо установить дополнительные пакеты, их наименование можно найти на [сайте](https://packages.msys2.org/groups/)

После завершения установки Qt Creator будет доступен по пути `C:\msys64\mingw64\bin`
